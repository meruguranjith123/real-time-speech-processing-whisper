# CSE 570 Project Report

This folder contains the complete LaTeX report for the Real-Time Assistive Speech Captioning System project.

## Files

- `main.tex` - Complete LaTeX document ready for Overleaf

## How to Use in Overleaf

1. Go to [Overleaf](https://www.overleaf.com)
2. Create a new project
3. Upload `main.tex` as the main file
4. Make sure you have the ACM template selected (sigconf, authorversion, nonacm)
5. Compile the document

## Notes

- The bibliography uses manual entries (not a .bib file) for simplicity
- All tables and formatting are included
- The document follows ACM format requirements
- You may need to adjust the bibliography style if Overleaf requires it
