## How to Use Demo Mode

1. **Open**: `http://localhost:8000`
2. **Click**: "Show Demo" button (pink button)
3. **Take Screenshot**: The page will show example outputs
4. **Click again**: Cycles through 3 different examples

## What Demo Shows

### Example 1: Sentence Repetition
- **Raw**: "I'm going to start with the first one. I'm going to start with the first one. I'm going to start with the first one. I'm going to start with the first one."

- **Stutters**: "Sentence repeated 4 times"
- **Predictions**: 3 next sentence suggestions

### Example 2: Word Repetition

- **Stutters**: Multiple word repetitions detected
- **Predictions**: Context-aware suggestions

### Example 3: Mixed Stuttering
